G.O.O.D

팀 프로젝트로 진행한 Spring 기반의 웹사이트입니다. 홈페이지 디자인은 마이리얼트립을 벤치마킹했으며, 저는 로그인, 프로필사진 업로드, 카카오 지도 API기반의 지도에 선형 정보 구현, 사용자의 위치정보를 기반으로 거리 계산, 기록하기 기능으로 기록된 코스를 글로 남기는 '나만의 코스' 게시판 등의 핵심 기능 구현을 맡았습니다. 또한 저를 제외하고 나머지 5인의 팀원들을 거의 도왔기때문에 프로젝트 전반의 코드나, 기능에 대하여 가장 잘 이해했으며 때문에 전체 프로젝트 디버깅을 맡았습니다.

기록하기 기능은 http 프로토콜 사용시 HTML5 Geolocation API 등 위치정보 사용제한때문에 실제 서버에서는 동작이 막혀있는 상태입니다.
